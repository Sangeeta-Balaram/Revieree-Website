# Revieree - Luxury Fragrances & Premium Cosmetics Website

A clean, production-ready version of the Revieree website with no admin panel. This version focuses purely on the customer-facing website experience.

## ✨ Features

- **Dynamic Hero Section**: Auto-rotating carousel of stunning product imagery
- **AI Product Finder**: Smart recommendations powered by AI
- **Responsive Design**: Mobile-first design with beautiful animations
- **Modern Tech Stack**: React 19, Vite, Tailwind CSS, Framer Motion
- **SEO Optimized**: Meta tags, semantic HTML, fast loading
- **Clean Architecture**: Component-based structure with lazy loading

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
